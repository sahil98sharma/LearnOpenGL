# OpenGL Learning Repository

## Overview
This repository contains code and exercises documenting my progression in learning graphics programming using OpenGL.

## Learning Resources
The following resources form the core curriculum for this repository:

* **Theory and Knowledgebase**: [LearnOpenGL](https://learnopengl.com/)
* **Video Tutorials**: [Victor Gordan's OpenGL Playlist](https://www.youtube.com/playlist?list=PLPaoO-vpZnumdcb4tZc4x5Q-v7CkrQ6M-)
* **Source Code and Exercises**: [Victor Gordan's GitHub Repository](https://github.com/VictorGordan/opengl-tutorials)
* **API Documentation**: [Official OpenGL Reference Pages (gl4)](https://registry.khronos.org/OpenGL-Refpages/gl4/)
